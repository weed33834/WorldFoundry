三步走：

1. `source WorldFoundry/tmp/worldfoundry_unified_env.sh` 并 `pip install -e ".[distribution_metrics]"`
2. `worldfoundry-eval metric show <id>` 确认 id / alias
3. 打开下面对应子页面，复制示例代码运行

只有 **built-in existing-results** 能走 CLI；其余 metric 用 Python `compute_*` 或 Scorer 类。

Metric 模块保持 lazy import：只有实际加载 feature extractor、多模态 scorer 或 API judge 时才需要 checkpoint。默认 cache 沿用 Hugging Face / Torch / torchvision layout：

```bash
export WORLDFOUNDRY_HFD_ROOT=${HF_HOME:-$HOME/.cache/huggingface}
export WORLDFOUNDRY_T2V_METRICS_CACHE_DIR=$WORLDFOUNDRY_HFD_ROOT
export WORLDFOUNDRY_CKPT_DIR=$HOME/.cache/worldfoundry/checkpoints
```
